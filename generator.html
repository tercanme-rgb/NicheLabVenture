Step 01
Get a free OpenRouter key
Go to openrouter.ai → Sign up → API Keys → Create key. It's free, no credit card required. You get 200 requests/day on the free models.
Step 02
Generate a report
Enter your niche in the left panel, click Generate. The AI writes the full report live. Takes 30–90 seconds. Review the output for accuracy before publishing.
Step 03
Copy the JSON
Switch to the JSON Editor tab. Click "Copy JSON". Open your reports.json file, add the object to the array (before the last ]), add a comma after the previous entry.
[ { ...existing report... }, { ...existing report... }, <-- paste new report here ]
Step 04
Push to GitHub
Save reports.json, commit, push. GitHub Pages deploys in ~30 seconds. Your dashboard and index page automatically show the new report.
git add reports.json git commit -m "Add report: [niche name]" git push
Note
Always review AI output
The model generates plausible-sounding data. Always fact-check signal numbers, competitor pricing, and TAM figures before publishing. The structure is correct — the numbers need verification.
